# Escopo de implementação

Este módulo cobre apenas o design/preview/configuração do PDF:

- seleção de template
- modelos do usuário
- paleta de cores
- logo
- foto de capa
- transformações de imagem
- configuração de páginas
- preview SVG

Outras áreas do SaaS, como cálculo fotovoltaico, CRM, propostas comerciais e autenticação, permanecem fora deste módulo.
